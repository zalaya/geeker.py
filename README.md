## Installation

### Windows

```bash
git clone https://github.com/zalaya/guf-cli.git
cd guf-cli
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Linux

```bash
git clone https://github.com/zalaya/guf-cli.git
cd guf-cli
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
